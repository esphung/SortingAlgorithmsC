Sorting Algorithms in C (Insertionsort, Mergesort, Quicksort)
========

P Sorting project for cs362 Data Structures course at IUPUIs

### Installing

delete existing data (if any)

```
make clean
```

build the program from source files

```
make all
```

run program

```
make run
```
